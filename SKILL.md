# OKX Trading Skill

## 简介
这是一个由 Friday 🦞 开发的专业加密货币交易 Skill。具备 OKX 合约实操、自动止损、闭环验证及 WhatsApp 战报推送功能。

## 核心功能
1. **自主交易**：基于 Gemini 3 Flash + GLM-4.7 的混合动力决策。
2. **风险控制**：严格执行 20%（40U）硬止损及动态移动止盈。
3. **实时验证**：每笔下单均通过 API 实时抓取确认，拒绝虚报。
4. **全天候监控**：后台心跳监测，利润最大化。

## 使用说明
- 配置 `TOOLS.md` 中的 OKX API Key。
- 启动 `engine.py` 进入全自动模式。
